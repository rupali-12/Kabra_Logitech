backened start command->

Features
Product Management:

Create new products with details like name, image, description, quantity, and unit price.
View a list of products with options to add them to the cart.
Cart Management:

Add products to the cart.
Update quantities of items in the cart.
View the list of items in the cart along with total price.
Technologies Used
Frontend:

React.js
Redux for state management
Redux Thunk for asynchronous actions
Bootstrap for styling
Backend:

Node.js with Express.js for API endpoints
MongoDB for data storage

Setup Instructions
Clone the repository:
git clone https://github.com/rupali-12/Kabra_Logitech.git
cd Assessment_kabra_logitech

Install dependencies:
npm install
